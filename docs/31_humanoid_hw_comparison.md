## 2. 판정 — 우리 다리 질량은 낮지 않다 (가설 기각)

**(a) 절대 세그먼트 질량 — 핵심 증거**
- **허벅지 5.10kg**: H1(4.95)·Cassie(5.52)와 사실상 동일. G1(1.70)의 3배, Digit(6.24)·GR-1(7.99)보다만 가벼움 → **정상, 오히려 무거운 축.**
- **정강이 2.82kg**: **H1(2.82)과 소수점까지 일치**, GR-1·G1(1.93)보다 무겁다 → **무거운 축.**
- **발 0.97kg**: G1(0.61)·Cassie(0.15)·Digit(0.58)보다 무겁고 H1(0.73)과 유사 → **정상.**

**(b) 다리 질량 비율 45.8%** — 피어 범위 41.1%(G1)~69.5%(Cassie), 중앙값 ~53%. 우리가 낮은 쪽이지만 이는 **상체 더미질량(base 28.1kg, 54%)이 무거워 분모가 큰** 탓이지 다리가 가벼워서가 아니다.

> **우리보다 무거운 H1·GR-1·Digit이 안 흐물거린다** = 흐물거림 원인이 질량이 아니라는 강한 방증.

## 3. 핵심 인사이트 — 질량 늘리면 흐물거림이 **더 나빠진다**

흐물거림(과진동)을 지배하는 건 질량이 아니라 **댐핑비 ζ**:

$$\zeta = \frac{c}{2\sqrt{k\,I}}, \qquad \omega_n=\sqrt{k/I}$$

**I(관성)가 분모 제곱근에 있다 → I가 커지면 ζ가 작아진다 = 더 출렁인다.** 가설이 제안한 "질량 추가" 방향이 정확히 역효과.

> [!warning] ★ 정정: 아래 PD는 초기 워크플로우(wr1cz09bp)에 **무릎 PD를 80/3으로 오기**한 채 돌려 나온 값이었음. **실제 무릎은 Kp 200 / Kd 5**(80/3은 ankle_pitch). 정성 결론(underdamped·질량무관)은 유지되나 — **정확한 per-joint PD·최종 댐핑값의 정본은 [[32_actuator_damping]]**.

무릎 실측 대입(**k=200, c=5**, 정정):

| 스윙 관성 I | ωₙ | ζ |
|---|---|---|
| 0.2 | 5.0 Hz | 0.40 |
| **0.3 (현재 추정)** | **4.1 Hz** | **0.32** |
| 0.5 | 3.2 Hz | 0.25 |

- 현재 무릎 **ζ≈0.32 → underdamped**(목표 0.7). c_crit=15.5, ζ=0.7엔 **Kd≈10.8 필요(현재 5의 2.2배).**
- **질량 추가(I↑)면 ζ 악화**(가설과 역효과).
- 엉덩이(k=200, c=5, I≈1.5): **ζ≈0.15 심한 underdamped**, ζ=0.7엔 Kd≈24.

**피어 대조 (정정):**
- **무릎 강성 200 = H1(200)과 동일, G1(150)·GR-1(120)보다 오히려 높음** → 강성은 정상. 문제는 **댐핑(5)**.
- GR-1 knee Kd 8 → ζ≈0.5, 우리 Kd 5 → ζ≈0.32 → **댐핑이 피어보다 낮아 더 underdamped.**
- 원인 = **낮은 댐핑**(강성 아님, 질량 아님).

## 4. 권장 조치 (→ 최종값 정본 [[32_actuator_damping]])

1. **댐핑 인상(즉효).** 무릎 Kd **5→11**(ζ→0.71), 엉덩이 Kd **5→24**, hip_yaw 5→6.5. 발목은 이미 ζ~0.7 유지. RS04 대형프레임 Kd 상한 100이라 가능.
2. **강성 유지.** 무릎 Kp 200은 피어 수준 — 손대지 않음(올리면 ζ↓).
3. **Action smoothing.** action rate penalty 강화·LPF·decimation 조정 → 2.6Hz PD가 못 따라가는 고주파 명령 제거.
4. **(비권장) 질량/관성 증가.** 이미 정상~무거운 축이고 **수학적으로 ζ를 더 낮춰 악화.** 건드리지 말 것.

> [!note] 솔직성 노트
> (1) 피어 **관성 텐서**는 공개 데이터 없음 → 질량으로 대리 판단. (2) 스윙 관성 0.3은 thigh 0.052+shin 0.033+거리² 추정치 — 실측 시 ζ 절대값은 달라져도 "underdamped"·"질량 추가는 역효과" 방향성은 불변. (3) Digit/Cassie Kd는 RL 게인이 아닌 물리 댐핑이라 직접 비교 제외.
