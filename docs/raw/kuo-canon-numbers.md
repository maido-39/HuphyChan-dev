# Kuo/Donelan/Ruina dynamic-walking — 검증 수치 (원문 대조)

> 출처: 워크플로우 w13anfb2o가 원문 fetch·검증. 인용주의 포함. wiki: [[Paperreview/kuo-donelan-dynamic-walking]].

- **외부 기계일 1→38W, 대사율 7→379W — 둘 다 걸음길이⁴** (r² 0.96/0.95, N=9) — Donelan, Kram & Kuo 2002 (JEB 205:3717) [link](https://journals.biologists.com/jeb/article/205/23/3717/9220/) · verified yes
- **선제 push-off가 mid-stance/hip 토크보다 4배 저렴**(collision 선차감) — Kuo 2002 (J Biomech Eng 124:113) [link](https://pubmed.ncbi.nlm.nih.gov/11871597/) · verified yes (초록)
- **push-off 15.4±2.6J / collision 12.4±3.1J / COM일 21.7J·step (@1.25m/s)**; 기계 COT **0.049**·순대사 COT **0.19**·순대사율 2.3 W/kg — Kuo, Donelan & Ruina 2005 (ESSR 33:88) [link](https://ruina.tam.cornell.edu/research/topics/locomotion_and_robotics/papers/energetic_consequences.pdf) · verified yes (Wayback 캐시 PDF)
- **전이 = 순대사비용 ~60–70% (≈2/3)** — *crude estimate*(가정효율 25%/−120%) — Kuo 2005 / Kuo&Donelan 2010 (Phys Ther 90:157, [PMC2816028](https://pmc.ncbi.nlm.nih.gov/articles/PMC2816028/)) · verified yes
- **평탄 COM → COT 0.18(인간 2배+); 강제 평탄화 → 대사율 +113%**; push-off 직전 적용 시 collision 손실 −75%; 발 roll-over q=0.30 — Kuo 2007 (Hum Mov Sci 26:617) [link](https://www.jychen.cn/papers/WalkModel/Kuo%20-%202007%20-%20The%20six%20determinants%20of%20gait%20and%20the%20inverted%20pend.pdf) · verified yes
- **hip 토크로 COM 방향전환 시 최대 4배 에너지 손실**; 외부 측면안정화 → 대사 ~9%↓ — Kuo&Donelan 2010 · verified yes

> ⚠ 인용주의: "60-70%"·"l⁴"는 **Kuo 2005/2007/2010에 인용**(Donelan 2002 *초록*엔 수치 없음, 본문 paywall). "9%"는 *외부* 측면안정화 실험치.
