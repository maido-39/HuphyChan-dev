## 2c. 학습 건강도 (TensorBoard: loss·수렴·낙상·보상항)
![tb](assets/2026-06-21_12-22-03_forefoot_cop_tensorboard.png)

- **수렴(noise_std)**: 0.32 → **0.27** (정체 ~)
- **mean_reward**: 0.6 → **39.7**, ep_len 최종 **984**
- **추종 error_vel_xy**: 최종 **0.545** (낮을수록 good), yaw 0.455
- **안정성 낙상률 3%** (base_contact 0.38 / time_out 13.00) (안정 ✅)
- **value loss 최종** 0.013, entropy -0.464, LR 3.8e-04
- **커리큘럼 vx 상한 최종** 2.00
- 정성 해석: **학습은 건강하게 수렴**. noise_std 0.32→0.27로 단조 감소 후 ~0.27 정체 = 정책이 탐색을 줄이고 안정화된 정상 수렴(미탐색·발산 신호 없음). value loss 0.013로 낮아 critic이 리턴을 잘 적합, entropy −0.464·LR 3.8e-04도 안정. 낙상률 3%(중반 스냅샷들은 <1%~0%)·ep_len 984/1000 ≈ 거의 완주 = **추락 없이 명령을 따름**. 단 error_vel_xy **0.545**는 부모(0.502)보다 *나쁨* — 신규 2항(특히 power_cot)이 추종 정확도를 약간 희생시킨 trade-off(에너지절감 우선). 커리큘럼 vx가 2.0(상한)에 일찍 도달해 정체한 것도 0.545의 일부 원인(최고난도 명령 상시 노출). **다음 튜닝**: 학습 자체는 잘 됐으므로 iter↑ 불필요 → 의도(toe 적재)가 실패했으니 **forefoot_cop 가중 강화** 또는 push-off-work 보상으로 전환(§5·아래 H-A 판정 참조).

- **정량** (측정 npz 있음: `logs/measure/forefoot_clip.npz`·`forefoot_unclip.npz`, §2의 RMS/max 표 기준):

  - **toe 사용도(핵심 지표)**: toe RMS 0.7/1.9 → **0.8/1.4 거의 불변** = 앞발 미적재 = **H-A 음성을 모터 레벨에서 확정**. forefoot_cop 보상 기여 +0.0072(총보상의 ~0.018%)로 gradient가 지배항에 묻힘.
  - **gait(추종·CoT)**: error_vel_xy 0.545(부모 0.502보다 악화), yaw 0.455. CoT는 power_cot 도입으로 knee·ankle_roll 연속토크 감소분만큼 개선 추정.
  - **낙상/진동**: 낙상 3%(중반 0~<1%)·ep_len 984 = 안정. 진동(>5Hz)·연결부 |F|/|M| 항목은 표준 npz에 포함되나 본 노트엔 미요약(필요시 `analyze_run.sh forefoot logs/measure/forefoot_clip.npz logs/measure/forefoot_unclip.npz` 재실행).
- **정성**:
  - **보행 자연스러움**: 수렴·낙상 0~3%·ep_len≈완주로 보행 자체는 안정적이고 매끄러움(noise_std 0.27 정체·value loss 0.013).
  - **실패모드**: 정책이 *앞발 굴림(rollover) 없이* 속도추종+에너지효율 gait를 발견 → 종말기 추진을 **toe가 아닌 ankle_pitch로 흡수**(ankle_pitch RMS↑가 증거). 즉 간접 CoP 보상이 toe 적재를 *우회*당함.
  - **의도한 변경의 효과(H-A)**: **음성**. forefoot_cop@0.5는 toe를 적재시키지 못함. power_cot은 *부수효과*로 ankle_roll/knee 연속토크는 줄였으나(부분 성공) 이는 H-A(앞발 적재)가 아닌 별개 이득. 진단: forefoot_cop이 게이트된 비율 보상이라 절대 기여가 본질적으로 작아 **가중↑만으로 충분치 않을 가능성**(형식 한계). Kuo 캐논([[Paperreview/kuo-donelan-dynamic-walking]]): toe 적재는 **push-off 일의 부산물**이므로 다음엔 push-off-work를 직접 보상하는 편이 원리적.
