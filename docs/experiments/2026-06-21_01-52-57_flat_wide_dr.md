## 5. 분석 (정성/정량)
- **정량(모터/gait)**: 이 run 디렉터리 아래 **측정 npz 없음**(`find … -name *.npz` 결과 0건) → 관절 토크/파워·toe 사용도·진동(>5Hz)·CoT의 **모터 정량분석은 보류**. 대신 TB 지표로 정량 대용:
  - 추종: error_vel_xy 0.532 / error_vel_yaw 0.552 (학습 중 max는 각 0.787 / 2.75 — yaw 1.57 확대 직후 큰 진동 후 수렴). std=0.5 기준 exp 보상으로 환산 시 track_lin 기여 +0.757·track_ang +0.712로 추종 자체는 살아있음.
  - 낙상: base_contact 종료비중 last 0.208(학습 초기 max 66 → 후반 안정), episode_length 987.7/1000 → 대부분 time_out 종료 = 낙상보다 완주 우세.
  - 안정화: Loss/value_function 0.0143로 수렴대(초기 max 0.79 대비 충분히 하강).
- **정성(스토리: stage-2 평지 + 넓은 DR)**:
  - **의도한 효과 = 달성**: 부모(전진 위주, 고정 명령)에서 vx [-1,2]·yaw ±1.57·강한 push·넓은 마찰로 분포를 넓혔고, curriculum_vel_x가 **2.0으로 완전 ramp**(min 1.0008→last 2.0000)된 채 episode_length를 유지 → 넓힌 명령·외란을 정책이 흡수.
  - **실패/주의 모드**: ① 추종 오차가 부모 대비 +60~80% 증가(넓은 명령의 가장자리, 특히 후진·고yaw에서 정확도 손실 추정). ② curriculum이 이미 상한 2.0에 붙어 더 끌어올릴 여지 없음 — intent의 "vx 2.5"는 **미적용**이고 [[16_dr_expansion]] 결론대로 **2.0 상한이 의도된 설계**(2.5는 Fr 0.80 달리기 영역이라 보행 PPO 불안정). 보고서 의도문구와 실제 cfg의 2.0 차이는 이 근거로 정합. ③ 외부 힘 DR은 [0,0]이라 실제 외력 로버스트성은 미검증(push 속도섭동만).
  - 종합: 절대 reward는 내려갔지만 **로버스트성 확보를 위한 의도된 trade-off**이며, 낙상 미증가·커리큘럼 완료로 다음 단계(발목 offload / rough terrain)의 안정적 init 체크포인트로 적합.
