- `track_ang_vel_z_exp`: **1.0 → 2.0** (2배). rough에서 지형이 yaw를 교란 → yaw 추종 가중을 키워 방향 유지 강화. 실제로 최종 기여 +1.1077로 1순위 보상이 됨(§2b).
- `feet_air_time`: **0.75 → 0.25** (1/3). flat 대비 rough에선 과도한 air-time 보상이 step-over/도약을 유발 → 낮춰서 지면접지 안정화. [[29_natural_gait_reward_hw]]·[[Paperreview/kuo-donelan-dynamic-walking]](swing 에너지 최소·접지 안정).
- `action_rate_l2`: **-0.005 → -0.008** (강화). 거친 지형 진동 입력에 대한 액션 채터링 억제 → 액추에이터 충실도/내구 [[28_reward_actuator_fidelity]].
- `dof_acc_l2`: **-1e-7 → -1.25e-7** (강화), `dof_torques_l2`: **-2e-6 → -1.5e-6** (완화). 가속 페널티는 올려 떨림 억제, 토크 페널티는 약간 풀어 계단·경사 극복에 필요한 순간 토크 허용 [[28_reward_actuator_fidelity]].
- **유지(신규 아님)**: `torque_soft_limit_ankle`(-0.01)·`torque_soft_limit`(-0.0025)는 부모와 동일 — 발목 포화 offload용([[23_toe_use_methods]]·docs/17·22)으로 stage-3에서 도입한 항을 rough로 그대로 계승.
- `upright`(+0.4024)·`base_height`(-0.1903)는 가중 동일이나 rough에서 기여 증가 → 지형 위 자세/높이 유지가 더 어려워졌음을 시사.
