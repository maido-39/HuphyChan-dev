# 오늘 연구 계획 — 2026-06-21 (12:00 → 21:00)

## 0. 최종 목적 재확인 (왜 이걸 하나)
> **RL이 만든 자연스럽고 효율적인 보행 → 관절·연결부의 실제 하중(토크·속도·구조 |F|/|M|) 측정 → "어느 모터/감속비/연결부를 얼마나 강하게 설계할지" 결정.** 
> 제약 = 물리적 **SIZE**(모터 선택). 보행 품질 = **인간형 + 에너지효율**. 즉 *학습은 수단, 산출물은 HW 사이징 근거*.

지금까지: 평지 보행 확립·발목offload·rough 전이. 측정으로 **ankle_roll 토크+속도 동시 포화 / knee 속도병목(1:3 과함)** 확인. 미해결: **toe 미적재**(간접 CoP보상으로 전환), **rough 미수렴**, **HW 사이징 확정**, **sim 하중 신뢰도**.
